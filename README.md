# 🎬 Anni Netflix – Netflix UI Clone

![Netflix Architecture](assets/images/Screenshot%202025-03-21%20154703.png)

Anni Netflix is a frontend web application that recreates the visual design and layout of the Netflix landing page.

The project focuses on replicating the modern streaming platform interface while demonstrating responsive web design and frontend development skills.

🌐 **Live Demo**  https://anni-twitter.netlify.app/

---

# 📌 Project Overview

Anni Netflix is a UI-focused clone inspired by the popular streaming platform **Netflix**.

The goal of the project is to practice building real-world website layouts using modern frontend technologies while recreating the look and feel of a professional streaming platform.

The application emphasizes:

- Clean streaming-platform UI design
- Responsive layout across devices
- Structured component-based frontend development
- Smooth browsing experience

---

# 🚀 Core Features

- 🎥 Netflix-style landing page design  
- 📺 Movie/series content sections  
- 🧭 Navigation bar similar to Netflix UI  
- 📱 Responsive design for desktop and mobile  
- ⚡ Smooth UI interactions and layout transitions

# ⚙️ How It Works

1. Users land on the homepage inspired by Netflix's interface.
2. The page displays featured content sections.
3. Movie and show thumbnails are presented in rows.
4. The responsive layout adapts across devices.

The project focuses mainly on **UI/UX replication rather than backend streaming functionality**.
